# 价格配置说明

## 📁 文件说明

这个文件夹包含两个页面共享的验证逻辑和价格配置：

- **`pricing-config.js`** - 价格计算配置文件（两个页面共用）

## 🎯 如何修改价格

**只需要修改 `pricing-config.js` 文件中的 `CFG` 对象**，两个页面会自动同步：

1. **估价表页面** (`E:\备份重要内容\米饭商行\估价表\index.html`)
2. **一填填表页面** (`E:\备份重要内容\米饭商行\一填填表\index.html`)

## ⚙️ 配置参数说明

### 基础比例（数值越大，租金越低）
- `base9: 41` - 9格保险基础比例
- `base6: 45` - 6格保险基础比例  
- `base24: 49` - 2格/4格保险基础比例

### 体力/负重加成
- `bonus_sta7_wgt7: 1` - 体力=7且负重=7时，比例减少1
- `bonus_sta6_wgt6: 0.5` - 体力=6且负重=6时，比例减少0.5
- `penalty_low: 1` - 体力≤3或负重≤3时，比例增加1

### 刀皮加成
- `bonus_chixiao: 1` - 有赤枭刀皮
- `bonus_knife_1: 0.5` - 有效刀皮1把
- `bonus_knife_2to3: 1` - 有效刀皮2-3把
- `bonus_knife_4to8: 1.5` - 有效刀皮4-8把
- `bonus_all_knife: 3` - 集齐全部10把刀皮

### 其他加成
- `bonus_weapon: 1` - 特殊武器皮肤（M7棱镜攻势/AS-Val-巨浪）
- `bonus_lingxiao: 2` - 凌霄戍卫干员红皮
- `bonus_hafu_201to300: 1` - 哈夫币201-300M惩罚
- `bonus_hafu_301plus: 2` - 哈夫币≥301M惩罚

### 体验卡加成
- `card24_X` - 2格/4格保险的体验卡加成
- `card6_X` - 6格保险的体验卡加成

## 💡 使用示例

例如，想让9格保险的租金更低（更优惠），可以将 `base9` 从 41 改为 42：

```javascript
const CFG = {
  base9: 42,  // 改这里
  base6: 45,
  base24: 49,
  // ... 其他配置
};
```

修改后保存文件，刷新两个页面即可生效！
